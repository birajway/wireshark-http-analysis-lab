Objective

To understand how HTTP transmits login requests in plaintext.

Tools

- Wireshark
- XAMPP
- DVWA
- Windows

Procedure

1. Installed XAMPP.
2. Configured DVWA.
3. Started Apache and MySQL.
4. Captured localhost traffic.
5. Filtered HTTP POST requests.
6. Examined HTTP Stream.

Result

Observed how HTTP form data is transmitted and learned how to inspect packets using Wireshark.

Disclaimer

This lab was performed on my own local machine for educational purposes.
